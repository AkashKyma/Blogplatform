# Changelog

## PAP-460 - Blog Platform

### Added
- Typography-focused blog platform built with Next.js 14 App Router, TypeScript, and Tailwind CSS
- Home page with featured post hero and recent post grid
- Blog archive page with live search and category filtering
- Dynamic blog post pages with article reader layout and related posts
- About page with author profile and social links
- Dark mode toggle with client-side theme persistence
- Static seed content and local image assets for posts and author profile
- Global not-found handling for unknown routes and missing blog slugs

### Notes
- Content is sourced from local static JSON data
- No external integrations or environment variables are required
- Ready for automated PR packaging after documentation review
